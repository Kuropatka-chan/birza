# Биржа DSM

Одностраничный сайт биржи с товарами, лотами на продажу и покупку, фильтрами и статистикой.

## Что реализовано

- Разбивка товаров по категориям и фильтр категорий на общем рынке.
- Переключение типа лотов (**продажа / покупка**) на общем рынке и в карточке товара.
- Привязка каждого лота к владельцу:
  - базовые лоты автоматически назначаются аккаунту `NotAHamster`;
  - пользовательские лоты в личном кабинете создаются от имени `Текущий пользователь`.
- Создание в личном кабинете как лотов на продажу, так и лотов на покупку.
- Сортировка лотов в карточке товара (цена, количество, владелец) с направлением сортировки.
- Статистика:
  - абсолютные значения по сделкам;
  - графическое отображение (бар-чарты) средней цены и количества предложений;
  - агрегация по периодам: **дни / недели / месяцы**.

## Запуск

1. Запустите локальный сервер в корне проекта:

   ```bash
   python3 -m http.server 8000
   ```

2. Откройте в браузере `http://localhost:8000`.

## Файлы

- `index.html` — структура интерфейса и элементы управления.
- `styles.css` — стили интерфейса, таблиц и графиков.
- `app.js` — загрузка данных, фильтрация/сортировка, сделки, создание лотов, статистика.
- `Биржа_с_ценами.txt` — исходные данные по товарам и базовым ценам.
