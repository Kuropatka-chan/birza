# Биржа DSM

В репозитории реализован одностраничный HTML-сайт биржи с товарами и ценами на основе файла `Биржа_с_ценами.txt`.

## Карта репозитория

- `index.html` — основной каркас интерфейса:
  - верхняя панель и переключение админ-доступа;
  - вкладки: **Общий рынок**, **Товар**, **Статистика**, **Личный кабинет**;
  - таблица общего списка товаров;
  - таблица предложений конкретного товара;
  - блок статистики;
  - форма создания личных лотов.
- `styles.css` — оформление интерфейса:
  - тёмная тема, таблицы, кнопки, вкладки;
  - адаптивность для меньших экранов;
  - стили для плейсхолдера иконок (`заглушка`);
  - стили выпадающих подсказок быстрого поиска.
- `app.js` — логика приложения:
  - загрузка и парсинг `Биржа_с_ценами.txt`;
  - генерация 3 предложений на каждый товар (базовая цена, -10%, +10%);
  - рендер общего списка и карточки товара;
  - покупка лота и накопление статистики;
  - общая статистика рынка и статистика по выбранному товару;
  - сортировка товаров по названию, цене и количеству;
  - быстрый поиск без учёта регистра + подсказки по началу слова;
  - личный кабинет для создания собственных лотов;
  - админ-режим (пароль `admin123`) для редактирования цены и количества предложений.
- `Биржа_с_ценами.txt` — исходные данные (цены и список товаров), на основании которых формируется начальный рынок.

## Как взаимодействовать с сайтом

1. Откройте `index.html` в браузере через локальный сервер (например, `python3 -m http.server`) — это нужно для загрузки текстового файла через `fetch`.
2. На вкладке **Общий рынок**:
   - используйте строку поиска (без учёта регистра);
   - выбирайте подсказку, если она появилась;
   - меняйте сортировку (название / цена / количество);
   - открывайте карточку товара или его статистику.
3. На вкладке **Товар**:
   - смотрите все предложения по продаже;
   - покупайте лоты кнопкой **Купить** справа в строке;
   - переходите к статистике выбранного товара.
4. На вкладке **Статистика**:
   - переключайтесь между общей статистикой рынка и статистикой выбранного товара.
5. На вкладке **Личный кабинет**:
   - создавайте собственные лоты (название товара, цена, количество);
   - можно выбрать существующий товар или ввести новый.
6. Админ-доступ:
   - нажмите **Включить админ-доступ**;
   - введите пароль `admin123`;
   - после включения во вкладке товара можно редактировать цены и количества предложений.

## Примечания

- Для каждого товара из файла автоматически создаются три рыночных предложения.
- Места под иконки товаров оставлены в виде визуальной заглушки с надписью `заглушка`.
